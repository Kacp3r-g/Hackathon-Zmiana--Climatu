Recycling Assistant Bot

Kacper Gorecki

Project Description

This project is a Discord bot designed to help users correctly dispose of waste.  
The main goal is to reduce recycling contamination and help people understand how to sort different types of waste properly.
The bot can answer questions, give tips, and even recognize waste items from images using AI.

Purpose

The purpose of this project is to:
Educate users about recycling
Help reduce environmental damage
Make waste sorting easier using technology
This bot could be used in schools, communities, or by anyone who wants to learn how to recycle correctly.

Features

Slash commands for easy interaction:
    /hello – greets the user
    /battery – how to dispose of batteries
    /plastic – how to recycle plastic
    /glass – how to recycle glass
    /paper – how to recycle paper
    /food – how to dispose of food waste
    /help – shows all commands
    /fact – gives a random recycling fact
    /tip – gives a useful recycling tip

AI Image Recognition:
User uploads an image
Bot detects the waste type using a trained model
Bot explains how to dispose of it correctly

How It Works

1. The user types a command (for example: /plastic)
2. The bot responds with instructions
OR
1. The user uses /scan and uploads an image
2. The AI model analyzes the image
3. The bot identifies the item and gives disposal advice
